# Agent Decisions Log

Running log of automated feature-build runs. Each run appends an entry here before opening its PR.
Entries should be terse. The PR body carries the detail, this file carries the pattern.

## Format

```
## YYYY-MM-DD HH:MM <timezone>

**Ticket:** FAM-###
**PR:** #<number> — <title>
**Branch:** ibarcroft/FAM-###

**Picked:** <feature>
**Why:** <one sentence on why this was next>
**Deferred:** <what was left for later, or "nothing">
**Notes:** <anything worth remembering — risky decisions, assumptions, uncertainty>
```
